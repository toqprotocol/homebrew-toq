# Homebrew tap for toq protocol

Official [Homebrew](https://brew.sh/) tap for [toq protocol](https://toq.dev).

## Install

```bash
brew install toqprotocol/toq/toq
```

## Install a specific version

```bash
brew install toqprotocol/toq/toq@0.1.0-alpha.1
```

## Upgrade

```bash
brew upgrade toqprotocol/toq/toq
```

## Uninstall

```bash
brew uninstall toq
brew untap toqprotocol/toq
```

## Other install methods

- Install script: `curl -sSf https://toq.dev/install.sh | sh`
- Cargo: `cargo install toq-cli`
- GitHub releases: [toqprotocol/toq/releases](https://github.com/toqprotocol/toq/releases)
